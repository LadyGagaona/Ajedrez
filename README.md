 Ajedrez - Parser de FEN


Este proyecto implementa un parser para cadenas FEN (Forsyth-Edwards Notation) que permite:

- Validar si una cadena FEN ingresada es correcta según la gramática oficial de ajedrez.

- Pintar un tablero de ajedrez en interfaz gráfica utilizando **Unicode** para representar las piezas.

- Mostrar mensajes claros de error si la cadena FEN no es válida, indicando el tipo de error y la fila afectada.  


## Lenguaje de programación
- **Java** (corretto 17 recomendado)

## Requisitos mínimos
- Sistema operativo: Windows, macOS o Linux
- Java corretto 17
- IDE recomendado: IntelliJ IDEA, Eclipse, o NetBeans
- Git para clonar el repositorio
